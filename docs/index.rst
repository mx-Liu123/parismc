PARIS: a multi-purpose MCMC sampler
-----------------------------------
PARIS is an advanced MCMC sampler. It supports parallel tempering,
flexible proposal moves, and can be extended to complex inference tasks.

Getting Started
---------------

Install:

.. code-block:: bash

   pip install parismc

Import:

.. code-block:: python

   from parismc import Sampler  # adjust to your API
